# commerce-changeset

A commerce changeset management application built with Next.js 16, Auth0, and Tailwind CSS v4.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Peer Dependencies

This project uses Next.js 16.2.1, React 19.2.4, and React-DOM 19.2.4 alongside `@auth0/nextjs-auth0@^4.16.0`. If you encounter peer dependency conflicts during `npm install`, use:

```bash
npm install --legacy-peer-deps
```

Alternatively, yarn handles peer dependencies more permissively:

```bash
yarn install
```
